# HH Job Assistant

Chrome-расширение для кандидата на hh.ru: помогает быстрее обрабатывать вакансии, отклики, резюме и переписку с работодателями.

## Бизнес-функции

- Запускает отклики прямо со страницы поиска вакансий hh.ru.
- Показывает прогресс обработки: текущее действие, счетчики, последние результаты и остановку процесса.
- Готовит персональные сопроводительные письма для вакансий, где нужен письменный ответ.
- Отвечает на вопросы работодателя в формах отклика.
- Помогает проходить сложные формы с текстовыми вопросами и вариантами выбора.
- Продолжает простые отклики, а вакансии, требующие ручного внимания, помечает отдельным результатом.
- Поднимает резюме на hh.ru, чтобы оно оставалось актуальным и заметным.
- Обрабатывает чаты с работодателями и готовит ответы для активных диалогов.
- Выделяет просьбы перейти во внешние каналы связи в отдельные отчеты.
- Оставляет управление у кандидата: запуск, остановка, поднятие резюме, обработка чатов и очистка отчетов выполняются вручную.

## Сценарий кандидата

1. Кандидат открывает подходящую выдачу вакансий на hh.ru.
2. Запускает обработку откликов из расширения.
3. Следит за прогрессом и последними результатами.
4. Останавливает процесс при необходимости.
5. Поднимает резюме, когда нужно поддержать видимость.
6. Обрабатывает чаты и просматривает подготовленные ответы или отчеты по внешним контактам.

## Безопасное поведение

- Не продолжает автоматические действия, если hh.ru требует ручную проверку.
- Не отправляет ответы работодателям, которые просят перейти во внешний канал.
- Пропускает вакансии, которые нельзя корректно обработать автоматически.
- Показывает пропуски и ошибки, чтобы кандидат мог решить, что обработать вручную.

## Границы продукта

Расширение сфокусировано на продуктивности кандидата на hh.ru:

- отклики;
- сопроводительные письма;
- ответы на вопросы работодателя;
- поднятие резюме;
- помощь с чатами;
- отчеты по внешним контактам;
- статус и результаты обработки.
